## Deskripsi Aplikasi

Vibe Music Player adalah aplikasi pemutar musik berbasis Android Native Java yang dirancang dengan tampilan modern dan pengalaman pengguna yang nyaman. Aplikasi ini memungkinkan pengguna memutar lagu, mengelola lagu favorit, melihat detail lagu, serta mengontrol musik melalui mini player dan full player.

Aplikasi ini dibuat sebagai project pembelajaran Android Studio dengan fokus pada implementasi UI modern, manajemen state playback, dan navigasi antar halaman.

## Tech Stack

- Java
- Android Studio
- XML Layout
- RecyclerView
- ViewPager2
- MediaPlayer
- CardView
- Material Design Components

## Fitur Utama

- Splash Screen saat aplikasi dibuka
