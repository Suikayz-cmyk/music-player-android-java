# Vibe Music Player

## Deskripsi Aplikasi

Vibe Music Player adalah aplikasi pemutar musik berbasis Android yang menyediakan fitur pemutaran lagu lokal menggunakan file MP3, sistem favorit, mini player interaktif, halaman detail player, serta tampilan modern dengan dark theme.

## Teknologi yang Digunakan

- Java
- Android Studio
- XML
- RecyclerView
- CardView
- Fragment
- MediaPlayer
- Bottom Navigation

## Fitur Utama

- Home Screen
  - Menampilkan daftar lagu menggunakan RecyclerView
  - CardView modern dengan cover album
  - Tombol play / pause langsung dari list lagu
  - Klik card lagu membuka halaman detail player

- Music Player System
  - Memutar lagu lokal dari asset MP3
  - Mini floating player di bagian bawah
  - Detail player screen
  - Sinkronisasi playback antara mini player dan detail player
  - SeekBar interaktif
  - Timer durasi lagu

- Favorite System
  - Menambahkan lagu ke favorit
  - Menghapus lagu dari favorit
  - Halaman Favorite khusus

- Account Screen
  - Foto profil
  - Nama dan email pengguna
  - Shortcut ke halaman Favorite
  - Informasi aplikasi

- UI / UX
  - Dark mode modern interface
  - Floating mini player
  - Album cover di setiap halaman
  - Navigasi bawah antar halaman

## Komponen Wajib yang Digunakan

- Fragment
- RecyclerView
- CardView
- Navigation Bar
- MP3 Assets
- PNG / Cover Image
- Icon Launcher
